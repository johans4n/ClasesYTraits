# ClasesYTraits :pencil:

ClasesYTraits es una colección de ejemplos de Clases y Traits en Scala.

## Requisitos :wrench:

* **sbt version: 1.4.7**
* **Scala version: 2.13.4**

## Implementación :hammer:

```
$ sbt console
$ run
```


## Traits :bulb:

* **Felino**


* **Rectangular**
